# Echart_block_Svelte

This program is the implementation of Echart block digram in Svelte

## Installation

Install the dependencies using:

```console
npm install
```

## Usage

You can run the program using:

```console
npm run dev
```

Or you can use the codesandbox link to check the program.

## Dependencies

echarts and svelte
